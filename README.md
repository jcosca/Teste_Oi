## Sobre este projeto

Este projeto é uma aplicação simples para um teste, feito com base no framework Laravel.

## Pré-requisitos

- MySql - V10.4.11-MariaDB.
- PHP - V7.4.1.

## Rodando aplicação

Após clonar o repositório, rode os seguintes comandos:

- npm install.
- php artisan serve.
